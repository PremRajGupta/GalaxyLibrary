**Print This & Keep on Admin's Desk! 📋**

---

## 🎯 WHERE IS IT?

```
Dashboard → Fees → Recent Payments Table
                   └─ Action Column
                      └─ [Mark Advance ⚡] Button
```

---

## ✅ QUICK STEPS

```
1. Dashboard → Fees
2. Find payment in "Recent Payments"
3. Click yellow ⚡ button
4. Enter/confirm Monthly Fee
5. Enter/confirm Start Date
6. Review preview (shows months + valid until)
7. Click "Mark as Advance"
8. Done! Status: VALID ✅
```

**Time Required:** 30 seconds per payment ⏱️

---

## 📊 WHAT IT DOES

```
INPUT:  Payment ₹1500, Fee ₹500
OUTPUT: 
  ✅ Months Covered: 3
  ✅ Valid Until: 04 Sep 2026
  ✅ Status: VALID (92 days)
  ✅ Auto-tracking: Activated
```

## ⚙️ HOW BACKEND CALCULATES

```
Admin Inputs:
├─ Amount: ₹1500
├─ Fee: ₹500
└─ Date: 04 June

Backend Calculates:
├─ Months: 1500 ÷ 500 = 3
├─ Expires: 04 June + 3 months = 04 Sep
└─ Days: 92 (automatic math)

Result Stored:
✅ monthsCovered: 3
✅ validUntilDate: 2026-09-04
✅ Status: "valid" (today < expiry)
```
